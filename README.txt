Save project folder to your Unity folder (default location: c:/users/<your name>/)
Open StartScene.unity in c:/users/<your name>/Rolling_Sky/Assets/Scenes/StartScene.unity via the Unity application.

Once the scene is opened, hit the "play" button at the top of the application.
When you are done playing, hit the "play" button again to reset the scene.
